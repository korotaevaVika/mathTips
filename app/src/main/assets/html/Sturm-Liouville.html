Ищем решения задачи Штурма-Лиувилля

Дано: ${X}''(x)+\lambda X(x)=0$

Решение

$p^{2}+\lambda =0 \rightarrow p^{2}=-\lambda$

Рассмотрим три случая

  • $-\lambda >0 \rightarrow X(x)=c_1e^{\sqrt{-\lambda} x} + c_2e^{-\sqrt{-\lambda} x} = c_1sh{\sqrt{-\lambda} x} + c_2ch{\sqrt{-\lambda} x}$
  • $-\lambda = 0 \rightarrow {X}''(x)= 0 \rightarrow X(x)=c_1x + c_2$
  • $-\lambda > 0 \rightarrow X(x)= c_1sin{\sqrt{\lambda} x} + c_2cos{\sqrt{\lambda} x}$

Собственные значения и функции для различных граничных условий

Для типа I-I $ \lambda _n = (\frac{\pi n}{l} )^{2}$ и $X_n (x)= sin(\frac{\pi n x}{l} )$

Для типа I-II $ \lambda _n = (\frac{\pi}{l}(n - \frac{1}{2} ))^{2}$ и $X_n (x)= sin(\frac{\pi x}{l} (n-\frac{1}{2}))$

Для типа II-I $ \lambda _n = (\frac{\pi}{l}(n - \frac{1}{2} ))^{2}$ и $X_n (x)= cos(\frac{\pi x}{l} (n-\frac{1}{2}))$

Для типа II-II $ \lambda _n = (\frac{\pi n}{l} )^{2} $и$ X_n (x)= cos(\frac{\pi n x}{l} )$;$ \lambda _0 = 0$ и $X_0 = 1 $
