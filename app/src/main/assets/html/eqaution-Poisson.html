Уравнение Пуассона в круге

Дано:

$\Delta U= \frac{\partial ^2U}{\partial x^2} + \frac{\partial ^2U}{\partial y^2} = f(x,y)$

$x^2+y^2< R^2$

$\Delta U(R)=q(x, y); x^2+y^2=R^2$

Решение

Перейдем к полярным координатам:

$\left\{\begin{matrix} x=rcos\varphi \\ y=rsin\varphi \end{matrix}\right.$

$\Delta U = U_{rr} + \frac{1}{r} U_{r} + \frac{1}{r^2} U_{\varphi \varphi}$

$f(x, y) = f(rcos\varphi, rsin\varphi)$

Преобразуем уравнение так, чтобы оно содержало только слагаемые типа $(a*r)^n cosk\varphi$ и/или $(b*r)^n sink\varphi$
