Приведение уравнения к каноническому виду

Дано: $a_{11}U_{xx} + 2a_{12}U_{xy} + a_{22}U_{yy} + a_{1}U_{x} + a_{2}U_{y} + U = f(x, y)$

Составим характеристическое уравнение

$a_{11} (dy)^2 - 2a_{12}dxdy + a_{22}(dx)^2 = 0 $

$\frac{D}{4} = a_{12}^2 - a_{11}a_{22}$

В зависимости от знака D, уравнение относится к одному из типов: гиперболический, параболический, эллиптический

  • D>0 Гиперболический тип.

    $\left\{\begin{matrix}dy = s*dx \\ dy =q*dx \end{matrix}\right.\Rightarrow \left\{\begin{matrix}dy = s*x + c_1 \\ y =q*x + c_2 \end{matrix}\right.$

    $ \left\{\begin{matrix} \xi =c_1=y-sx \\ \mu =c_2=y-qx \end{matrix}\right.$, где $\xi$ и $\mu$ общие интегралы

  • D=0 Параболический тип. $\xi$ общий интеграл, $\mu$ - это x или y
  • D<0 Эллиптический тип. $\xi$ и $\mu$ - действительная и мнимая часть одного из решений

Далее необходимо выразить частные производные и подставить в эти выражения полученные ранее $\xi$ и $\mu$

$U_{x} = U_{\xi }\xi_x + U_{\mu }\mu_{x}$

$U_{y} = U_{\xi }\xi_y + U_{\mu }\mu_{y}$

$U_{xx} = (U_{\xi }\xi_x)_x + (U_{\mu }\mu_{x})_x =$

$ = (U_{\xi\xi }\xi_x \xi_x + U_{\xi\mu}\xi_x \mu_x + U_{\xi }\xi_{xx}) + (U_{\mu\xi}\mu_x \xi_x + U_{\mu\mu}\mu_x \mu_x + U_{\mu}\mu_{xx})$

$U_{xy} = (U_{\xi }\xi_x)_y + (U_{\mu }\mu_{x})_y= (U_{\xi\xi }\xi_x \xi_y + U_{\xi\mu}\xi_x \mu_y + U_{\xi }\xi_{xy}) + (U_{\mu\xi}\mu_x \xi_y + U_{\mu\mu}\mu_x \mu_y + U_{\mu}\mu_{xy}) $

$U_{yy} = (U_{\xi }\xi_y)_y + (U_{\mu }\mu_{y})_y =$

$ = (U_{\xi\xi }\xi_y \xi_y + U_{\xi\mu}\xi_y \mu_y + U_{\xi }\xi_{yy}) + (U_{\mu\xi}\mu_y \xi_y + U_{\mu\mu}\mu_y \mu_y + U_{\mu}\mu_{yy})$
